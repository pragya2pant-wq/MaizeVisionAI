# 🌽 MaizeVision AI

An AI-powered web application for detecting maize leaf diseases using Deep Learning and Streamlit.

## Features

- 🌿 Detects four maize leaf classes:
  - Blight
  - Common Rust
  - Gray Leaf Spot
  - Healthy
- 🧠 Supports multiple Deep Learning models
  - Custom CNN
  - VGG16
- 📊 Interactive prediction confidence
- 📈 Model performance dashboard
- 📉 Confusion matrix visualization
- 📚 Disease information library
- 📝 Prediction history

## Technologies Used

- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Pillow
